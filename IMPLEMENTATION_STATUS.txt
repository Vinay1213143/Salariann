================================================================================
FALLBACK BEHAVIOR
================================================================================

Cost of Living Fallback:
  1. Try Numbeo API (if NUMBEO_API_KEY set)
  2. Try Teleport API (free, no key needed)
  3. Use FALLBACK_COSTS (hardcoded data)

Jobs Fallback:
  1. Try JSearch API (RapidAPI)
  2. Try Adzuna API (if configured)
  3. Use Database jobs

Response Indicators:
  "source": "live"      → Data from external API
  "source": "fallback"  → Data from hardcoded backup

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Getting "fallback" instead of "live"
  Solution:
    1. Check NUMBEO_API_KEY in .env
    2. Verify API key is correct
    3. Test API directly
    4. Check Numbeo API status
    5. Fallback is normal if API is down!

Issue: Slow responses
  Solution:
    1. Enable caching: CACHE_ENABLED=true
    2. Increase TTL: CACHE_TTL=86400
    3. Check API status
    4. Use fallback data

Issue: "Unable to fetch cost of living data"
  Solution:
    1. Check network connectivity
    2. Verify API keys
    3. Check API status pages
    4. Ensure fallback data exists
