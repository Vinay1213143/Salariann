## 🔑 Key Concepts

### Screens
- **Home Screen**: Job discovery with search and filtering
- **Companies Screen**: Company directory and search
- **App Shell**: Main navigation container with responsive layouts

### Widgets
- **Filter Panel**: Advanced filtering controls
- **Job Card**: Individual job listing component
- **Affordability Badge**: Color-coded salary suitability indicator

### Providers
- **AuthProvider**: User authentication and session
- **JobsProvider**: Job data, filtering, and search
- **CompaniesProvider**: Company data and search
- **UserProvider**: User profile and preferences

### Navigation
- **GoRouter**: Modern navigation with named routes
- **StatefulShellRoute**: Main navigation with branch preservation
- **5 Tabs**: Home, Companies, Notifications, Messages, Profile

### Responsive Design
- **Mobile** (< 600px): Bottom navigation bar
- **Tablet** (600-1024px): Left navigation rail
- **Desktop** (> 1024px): Extended rail + right sidebar
