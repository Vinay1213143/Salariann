## 🚀 STEP-BY-STEP (5 MINUTES)

### Step 1: Create New Project (1 min)
1. Go to https://www.overleaf.com/
2. Click **"New Project"**
3. Select **"Blank Project"**
4. Name: `Salariann Project Report`
5. Click **"Create"**

### Step 2: Upload Files (2 min)
1. Click **"Upload"** button (top left)
2. Click **"Select a .zip file"** OR drag & drop files
3. Select all 4 files from your computer
4. Wait for upload to complete

### Step 3: Set Main File (1 min)
1. Click **Menu** (top left, three lines)
2. Scroll down to **"Main document"**
3. Click dropdown and select **`OVERLEAF_MASTER_FILE.tex`**

### Step 4: Compile (1 min)
1. Click **"Recompile"** button (or Ctrl+Enter)
2. Wait 1-2 minutes
3. PDF appears on right side!
4. Click **"Download PDF"** to save
