## 🚀 How It Works

1. **User Signs In** → Profile page loads
2. **Avatar Generated** → First letter of email
3. **User Info Displayed** → Name, headline, email
4. **Edit Profile** → Update name and lifestyle
5. **Save Changes** → Updates in database
6. **Contribute** → Navigate to contribution pages

---

## 📝 User Actions

- **Open to**: Shows availability (placeholder)
- **Sign Out**: Logs out and returns to home
- **Edit Name**: Update display name
- **Select Lifestyle**: Choose Single or Family
- **Save Changes**: Update profile
- **Add Review**: Navigate to review page
- **Contribute Salary**: Navigate to salary page
- **Share Interview**: Navigate to interview page
