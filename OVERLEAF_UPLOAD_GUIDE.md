## 🚀 STEP-BY-STEP OVERLEAF SETUP

### Step 1: Create New Project on Overleaf
1. Go to https://www.overleaf.com/
2. Click **"New Project"**
3. Select **"Blank Project"**
4. Name it: `Salariann Project Report`
5. Click **"Create"**

### Step 2: Upload Files
1. In your Overleaf project, click **"Upload"** (top left)
2. Select **"Select a .zip file"** or upload files individually
3. Upload all 4 files listed above
4. Wait for upload to complete

### Step 3: Set Main File
1. In Overleaf, click the **Menu** icon (top left)
2. Find **"Main document"** section
3. Select **OVERLEAF_MASTER_FILE.tex** as the main file

### Step 4: Compile
1. Click the **"Recompile"** button (or Ctrl+Enter)
2. Wait for compilation to complete
3. Your PDF will appear on the right side!

## 🔧 TROUBLESHOOTING IN OVERLEAF

### Issue: "File not found" error
**Solution**: Make sure all 4 files are uploaded and in the same folder

### Issue: Diagrams not showing
**Solution**: Overleaf automatically compiles TikZ. Wait a moment for rendering.

### Issue: Special characters (₹) not displaying
**Solution**: Overleaf uses UTF-8 by default. Should work fine. If not, use xelatex compiler.

### Issue: Compilation takes too long
**Solution**: This is normal for a 100-page document. Wait 1-2 minutes.

### Issue: "Undefined control sequence" error
**Solution**: All packages are included. Check that all 4 files are uploaded.
